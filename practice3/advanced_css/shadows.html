Text-shadow effect!

The CSS text-shadow property applies shadow to text. In its simplest use you only specify the:

  • horizontal shadow (2px)
  • vertical shadow (2px)
  • color to the shadow (black)
  • blur effect to the shadow (4px)

Multiple text-shadows

To add more than one shadow to the text, you can add a comma-separated list of shadows. The following example shows a red and blue neon glow shadow:

  • text-shadow: 0 0 3px #FF0000, 0 0 5px #0000FF;

result

The following example shows a white text with black, blue, and darkblue shadow:

  • color: white; text-shadow: 1px 1px 2px black, 0 0 25px blue, 0 0 5px darkblue;
